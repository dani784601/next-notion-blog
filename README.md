# Next.js & Notion API Blog

## 프로젝트 소개

인터넷 강좌를 통해 배운 Next.js와 Notion API를 활용하여 개인 블로그를 제작하였습니다. 노션을 CMS로 활용하여 쉽게 포스트를 업데이트할 수 있습니다.

## 사용한 기술 스택

- Next.js
- Notion API
- Vercel (배포)

## 주요 기능

- 노션의 페이지를 블로그 포스트로 변환
- 태그별 포스트 분류
- 태그 검색 기능

## 설치 및 실행방법

```bash
git clone next-notion-blog
cd next-notion-blog
npm install
npm run dev
```
